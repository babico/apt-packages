Quick Install — 1.4.6

amd64 (one-liner)

curl -fsSL https://babico.github.io/rustdesk-repo/rustdesk-apt.gpg | sudo gpg --dearmor -o /usr/share/keyrings/rustdesk.gpg
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/rustdesk.gpg] https://babico.github.io/rustdesk-repo stable main" | sudo tee /etc/apt/sources.list.d/rustdesk.list
sudo apt update && sudo apt install rustdesk

Step-by-step Setup

1

Import the signing key

curl -fsSL https://babico.github.io/rustdesk-repo/rustdesk-apt.gpg | sudo gpg --dearmor -o /usr/share/keyrings/rustdesk.gpg
2

Add the repository — choose your architecture

amd64 (x86-64):

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/rustdesk.gpg] https://babico.github.io/rustdesk-repo stable main" | sudo tee /etc/apt/sources.list.d/rustdesk.list

arm64 (AArch64 / Raspberry Pi 64-bit):

echo "deb [arch=arm64 signed-by=/usr/share/keyrings/rustdesk.gpg] https://babico.github.io/rustdesk-repo stable main" | sudo tee /etc/apt/sources.list.d/rustdesk.list

armhf (ARMv7 32-bit):

echo "deb [arch=armhf signed-by=/usr/share/keyrings/rustdesk.gpg] https://babico.github.io/rustdesk-repo stable main" | sudo tee /etc/apt/sources.list.d/rustdesk.list
3

Install

sudo apt update && sudo apt install rustdesk
4

Stay updated

sudo apt upgrade rustdesk

All future releases are picked up automatically by apt upgrade.

Install a specific version

Install exact version

# Any version from the table below
sudo apt install rustdesk=1.4.5

Hold (prevent auto-upgrade)

sudo apt-mark hold rustdesk
sudo apt-mark unhold rustdesk   # re-enable

Downgrade

sudo apt install rustdesk=1.4.4
